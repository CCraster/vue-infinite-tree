- [x] 虚拟滚动 tree

- [x] 节点删除 - 快捷键 delete
- [x] 节点修改
- [x] 节点复选框
  - [x] 功能可选
  - [] 可设置节点不可选中
- [] 单个节点选中
- [] 可设置默认选中复选框的树节点
- [] 可设置默认选中的树节点
- [] 可设置默认展开的树节点
- [x] 可设置是否默认展开所有树节点

- [] 动画 - 节点展开/折叠、删除、增加
- [] 事件系统

- [] 节点可拖拽
